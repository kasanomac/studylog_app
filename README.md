# The-View
勉強時間を記録するサイトです。<br>
メモや写真も一緒に記録できます。<br>
他の人の記録を見ることができます。<br>
「ゲストログインとして利用する」リンクを押すと新規登録なしでログインできます。

# 使用技術
- Ruby 3.3.0
- Ruby on Rails 7.1.3.2
- Postgresql 14.12
- Puma
- RSpec
 
# 機能一覧
- ユーザー登録、ログイン機能（devise）
- ユーザー情報の編集機能
- 記録投稿機能
 - 画像投稿（Active Strage）
- 記録詳細表示機能
- 記録一覧表示機能
- ページネーション機能（kaminari）
- ユーザー検索機能（ransack）
